# 资源管理器交互逻辑文档

> 本文档描述 Mede IDE 资源管理器（双列文件管理器）的完整交互逻辑，用于对照期望进行验收。

---

## 一、整体布局

- **左列**：始终显示，默认加载项目根目录
- **右列**：通过顶部工具栏「展开/收起」按钮显隐，独立导航
- **顶部工具栏**：显示当前活跃窗口的目录名 + 展开/收起按钮 + 选择计数 + 取消按钮
- **底部工具栏**：双模式（普通导航 / 批量操作），基于活跃窗口动态切换
- 两列之间用竖线分隔，各自独立滚动

---

## 二、窗口活跃状态

- 用户最后一次点击/滑动/长按的窗口即为**活跃窗口**
- 底部工具栏的所有导航操作（返回上级、前进、新建、回到根目录）均作用于**活跃窗口**
- 顶部路径面包屑显示**活跃窗口**的当前目录名
- 左右两列文件列表中的 `..`（返回上级）各自独立，不受活跃窗口影响

---

## 三、文件导航

### 3.1 进入子目录
- 点击目录（文件夹图标）→ 当前列直接加载子目录内容
- 左列导航只更新左列；右列导航只更新右列

### 3.2 返回父目录
- 每列顶部自动显示 `..` 条目（仅当前不在根目录时）
- 点击 `..` → 当前列返回上级目录
- 该操作与活跃窗口无关，各自列独立生效

### 3.3 打开文件
- 点击非目录文件 → 在编辑器中打开该文件

---

## 四、多选交互

> **核心原则**：多选**不跨窗口**，仅在**活动窗口**内操作

### 4.1 进入多选模式
- **左右滑动任意文件**（滑动距离 > 60px）→ 进入多选模式，选中该文件，设为 anchor

### 4.2 范围多选（两次滑动）
- 首次滑动文件 A → 选中 A（anchor = A）
- **再次滑动**文件 B（同窗口）→ 范围选中 A 与 B 之间的**全部文件**
- 范围多选仅在**连续两次都是滑动**时触发

### 4.3 单文件切换（点击或滑动后点击）
- **点击**文件 → 只切换当前文件的选中状态，**不触发范围选中**
- 点击后更新 anchor = 当前文件
- **点击后再滑动**另一文件 → 因上次不是滑动，只追加选中当前文件，不范围选中

### 4.4 跨窗口行为
- **左列有多选（≥2）**，去右列滑动/点击 → **清空左列所有选中**，右列单选当前文件
- **左列只有1个选中**，去右列滑动/点击 → **不清空左列**，右列追加选中1个（全局共2个，分属两列，用于对比）
- **双列各选1个后** → 全局锁定为2个，**其它文件不可再选中**
- 取消任一选中 → 解除锁定，恢复正常多选

### 4.5 取消多选
- 顶部工具栏点击「取消」→ 清空所有选中，退出多选模式
- 底部批量操作栏点击「取消」→ 同上

---

## 五、长按菜单（上下文菜单）

- 长按任意文件弹出菜单
- 菜单项根据当前状态动态显示：

| 场景 | 显示项 |
|------|--------|
| 单选（未展开第二列或右列未加载） | 打开文件、添加到对话、新建文件、新建文件夹、重命名、删除、复制名称、复制路径、查看信息、压缩、打开为项目 |
| 双列展开 | 额外显示「复制到左侧/右侧目录」「移动到左侧/右侧目录」 |
| 多选（≥2） | 禁用：打开文件、重命名、查看信息、复制到左/右、移动 到左/右、打开为项目；<br>启用：批量压缩 X 项、批量删除 X 项 |

- 复制/移动为**一步直达**：无需粘贴，直接跨窗口完成
- **单列模式下不显示**复制/移动选项（没有目标窗口），展开第二列后才显示

---

## 六、底部工具栏 — 普通导航模式

> 当未选中文件时显示

| 按钮 | 行为 |
|------|------|
| ← 返回上级 | 活跃窗口返回父目录 |
| → 前进 | 基于活跃窗口独立历史栈前进 |
| + 新建 | 在活跃窗口当前目录下弹出新建文件/文件夹对话框 |
| ⇄ 同步 | 将**非活跃列**同步为**活跃列**的当前路径（例如操作左列时右列跟过来） |
| ↑ 回到根目录 | 活跃窗口跳回项目根目录 |

### 6.1 历史栈
- 左右窗口各自拥有独立的前进/后退历史栈
- 进入子目录时自动压栈
- 返回上级/根目录不会丢失前进记录

---

## 七、底部工具栏 — 批量操作模式

> 当选中至少一个文件时，底部工具栏自动切换为此模式

| 按钮 | 行为 |
|------|------|
| 全选 | 选中活跃窗口当前目录下所有文件 |
| 反选 | 反转活跃窗口当前目录下的选中状态 |
| 取消 | 清空所有选中，退出多选 |
| 对比 | 当全局恰好选中 2 个文本文件（< 5MB）时启用，调用 Diff |
| 合并 | 同上条件，将第二个文件内容追加到第一个 |

### 7.1 Diff/Merge 规则
- 必须恰好选中 **2 个文件**
- 两文件都必须是文本类型（kt/java/txt/md/xml/json/... 等）
- 单文件大小 < 5MB
- **支持跨列对比**：左列选中一个 + 右列选中一个 → 可对比不同目录下的文件

---

## 八、顶部工具栏

- 显示**活跃窗口**当前目录名
- 「展开/收起」按钮：切换右列显隐
- 选中文件时显示：`X 已选 | 取消`
- 点击取消 → 清空选中并退出多选

---

## 九、压缩与压缩包浏览

### 9.1 压缩
- 单文件长按 → 压缩该文件
- 多选长按 → 显示「批量压缩 X 项」
- 弹出压缩对话框：文件名、格式（zip/tar/gz）、压缩级别（0-9）、密码开关 + 输入
- 使用 zip4j 支持密码保护 ZIP

### 9.2 压缩包浏览
- 点击压缩文件 → 在当前列直接浏览压缩包内容（不进编辑器）
- 压缩包内文件同样支持**滑动多选**
- 长按单文件 → 「解压到左侧/右侧目录」
- 多选长按 → 「批量解压到左侧/右侧 X 项」

---

## 十、同步功能详解

- 场景：需将当前目录文件复制到同目录下的子文件夹
- 操作步骤：
  1. 在左列浏览目标目录
  2. 点击同步 → 右列同步为左列路径
  3. 在右列进入子文件夹
  4. 在左列选中文件 → 长按「复制到右侧」→ 一步完成

---

## 十一、空文件夹处理

- 目录为空时显示「空文件夹」提示文字
- 若不在根目录，同时显示 `..` 返回上级条目
